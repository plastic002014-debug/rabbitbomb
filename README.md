# 🐰 돈먹는 토깽이

바구니를 든 토끼로 동전을 받아 점수를 올리는 캐주얼 게임!

## 🎮 플레이 방법

- **← →** 키 (PC) 또는 화면 좌우 버튼 (모바일)으로 토끼 이동
- 위에서 떨어지는 **동전**을 받으면 점수 획득
- **💣 폭탄**을 3번 받으면 게임 오버

## ✨ 게임 특징

- 점수가 높아질수록 낙하 속도 · 폭탄 확률 상승
- **20콤보** 달성 시 보너스 타임 발동 (폭탄 없음!)
- 🥇🥈🥉 이니셜 3글자 스코어보드 등록
- BGM · 효과음 · 햅틱 진동(only AOS) 지원 

## 🚀 GitHub Pages 배포

1. 이 저장소를 GitHub에 업로드
2. Settings → Pages → Branch: `main` / `/ (root)` → Save
3. `https://<유저이름>.github.io/<저장소명>/` 으로 접속

## 📁 파일 구조

```
index.html   - HTML 구조
style.css    - 스타일시트
script.js    - 게임 로직
```
